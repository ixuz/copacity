# Copacity

## Prerequisites

- Docker
- WSL
- Visual Studio Code
  - WSL (Extension)
  - Dev Containers (Extension)

## Build

Clone and open this repository with **Visual Studio Code** and when prompted, click the _Reopen in Dev Container_ button.

You may need to adjust the ownership of the `/workspace` folder before launching a build task.

```sh
$ sudo chown -R vscode:vscode /workspace
```
